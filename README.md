# 🕹️ Run for Values

"Run for Values" is a mini-game inspired by the classic offline Chrome dinosaur game — but with a powerful twist. Built with **HTML**, **CSS**, and **JavaScript**, the game transforms company values into an interactive, engaging experience.

## 🌟 Concept

At Orbyta, we’re driven by five core values:

- **Uniqverse**
- **Belongship**
- **Inspirelead**
- **Mindshift**
- **Ownability**

🎮 *What if you could collect these values in a game?*

In *Run for Values*, the player navigates through falling obstacles and collects floating value tokens. The mission is simple: **survive**, **dodge**, and **collect all five values** to win the game.

---

## 🔧 Technologies Used

- **HTML** – Structure
- **CSS** – Styling and animations
- **JavaScript** – Game logic and interactivity

---

## 📱 Features

- Vertical layout optimized for mobile screens  
- Smooth animations and simple controls (tap or arrow keys)  
- Word collection mechanic  
- Obstacle collision detection  
- Victory screen with celebration (coming soon: confetti bomb!)  
- Lightweight and fast loading
